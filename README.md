# Landing Page - Desenrola AI

Landing page para o evento "Desenrola AI" em Fortaleza/CE, previsto para outubro de 2025.

## Sobre o Projeto

Esta landing page foi criada para o pré-lançamento do evento Desenrola AI, focando em:
- Construção de expectativa e suspense
- Captura de leads para lista de espera
- Design moderno e responsivo
- Preparação para futura evolução em página de vendas

## Funcionalidades

- **Contagem Regressiva**: Cronômetro dinâmico até outubro de 2025
- **Formulário de Leads**: Captura nome, e-mail e telefone
- **Armazenamento Local**: Simulação com localStorage (preparado para backend futuro)
- **Design Responsivo**: Otimizado para desktop e mobile
- **Navegação Suave**: Scroll suave entre seções

## Estrutura dos Arquivos

```
/
├── index.html      # Estrutura HTML da página
├── styles.css      # Estilos CSS responsivos
├── script.js       # JavaScript para funcionalidades
└── README.md       # Este arquivo
```

## Como Usar

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Navegue pelas seções usando o menu ou scroll
3. Teste o formulário de inscrição na seção "Lista de Espera"
4. Verifique o console do navegador para ver os leads armazenados
